# 校园信息化平台 - Campus Information Platform (CIP)

#### 本项目计划制作一个更加智能的校园信息化平台，基于XJTLU校园设计，能够推广到其他大学。

## 主要亮点：
1.自定义主页，提供16+小组件，能够自行选择（课表、日历、邮箱等）。

5.提供个人主页，可以自定义展示内容，作为个人能力的展示。
